# Page-d-accueil
## Projet : 

Créer une page web complète à partir d'une maquette fournie, en suivant les instructions du cours sur The Odin Project.

Ce projet consiste à reproduire une page web à partir d'une maquette, sans viser la perfection au pixel près. L'objectif est de positionner différents éléments de la page de manière approximative, en utilisant les compétences acquises dans le cours.

## Technologies utilisées : 

* HTML 
* CSS (avec Flexbox pour la mise en page)

## Instructions
 
 1- Reproduire la page web à partir de la maquette fournie.
 2- Utiliser HTML et CSS avec Flexbox pour structurer et styler la page.
 3- Ne pas se soucier de la perfection au pixel près.
